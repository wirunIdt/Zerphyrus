ภาพรวม If Clauses

มองเห็นความสัมพันธ์ของประโยคเงื่อนไขทั้ง 4 ประเภทในภาพเดียว

Structure

โครงสร้าง ทั้ง 4 ประเภท

ทำความเข้าใจโครงสร้างและความหมายของแต่ละ conditional

Zero Conditional

ความจริงทั่วไป / กฎธรรมชาติ

ใช้พูดถึงสิ่งที่เป็นความจริงเสมอ ผลลัพธ์ที่เกิดขึ้นแน่นอนเมื่อมีเงื่อนไขนั้น

If + S + V1 , S + V1
e.g. If you heat ice, it melts.
First Conditional

เป็นไปได้จริงในอนาคต

ใช้พูดถึงสถานการณ์ที่อาจเกิดขึ้นได้จริงในอนาคต มีความเป็นไปได้สูง

If + S + V1 , S + will + V1
e.g. If it rains, I will stay home.
Second Conditional

สมมติ / ไม่น่าเป็นไปได้

สมมติสถานการณ์ที่ไม่เป็นจริง หรือแทบไม่มีทางเกิดขึ้นในปัจจุบัน

If + S + V2/were , S + would + V1
e.g. If I were rich, I would travel.
Third Conditional

สมมติอดีต / เสียดายที่ผ่านไปแล้ว

สมมติว่าถ้าเหตุการณ์ในอดีตเกิดต่างออกไป ผลลัพธ์จะเป็นอย่างไร

If + S + had+V3 , S + would have+V3
e.g. If I had studied, I would have passed.

📊 ตารางสรุปโครงสร้าง

TypeIf ClauseMain Clauseการใช้

FirstIf + V1will + V1อนาคตที่เป็นไปได้
SecondIf + V2/werewould + V1สมมติปัจจุบัน

How to Use

วิธีเลือกใช้ อย่างถูกต้อง

คำถาม 4 ข้อนี้จะช่วยให้คุณเลือก conditional ได้ถูกต้อง

🌍

ความจริงทั่วไปหรือไม่?

ถ้าพูดถึงกฎธรรมชาติ หรือสิ่งที่เป็นจริงเสมอ → Zero
"If you drop a ball, it falls."

🔮

อนาคต เป็นไปได้จริงไหม?

ถ้าสถานการณ์มีโอกาสเกิดขึ้นจริงในอนาคต → First
"If I have time, I will help."

💭

สมมติ ไม่เป็นความจริงตอนนี้?

ถ้าเป็นเรื่องสมมติที่ไม่น่าเกิดขึ้น → Second
"If I were a bird, I would fly."

⏰

สมมติอดีต ที่ผ่านไปแล้ว?

ถ้าสมมติเหตุการณ์อดีตที่เปลี่ยนไม่ได้ → Third
"If I had known, I would have called."

💡
Tips: ใน Second Conditional ใช้ "were" แทน "was" กับทุก subject — "If I were you…" / "If he were here…" เพราะเป็น Subjunctive Mood
Examples

ตัวอย่าง ประโยคทุกรูปแบบ

ประโยคบอกเล่า ปฏิเสธ และคำถาม ของทั้ง 4 conditional

Zero Conditional
If you heat water to 100°C, it boils.
🇹🇭 ถ้าคุณให้ความร้อนน้ำถึง 100°C มันจะเดือด
If dogs eat chocolate, they get sick.
🇹🇭 ถ้าสุนัขกินช็อกโกแลต มันจะป่วย
First Conditional
If it rains tomorrow, I will stay home.
🇹🇭 ถ้าฝนตกพรุ่งนี้ ฉันจะอยู่บ้าน
If she studies hard, she will pass the exam.
🇹🇭 ถ้าเธอเรียนหนัก เธอจะสอบผ่าน
Second Conditional
If I were rich, I would travel the world.
🇹🇭 ถ้าฉันรวย ฉันคงจะท่องเที่ยวรอบโลก
If he had a car, he would drive to work.
🇹🇭 ถ้าเขามีรถ เขาคงขับรถไปทำงาน
Third Conditional
If I had studied, I would have passed the test.
🇹🇭 ถ้าฉันตั้งใจเรียน ฉันคงสอบผ่านไปแล้ว
If she had left earlier, she would have caught the bus.
🇹🇭 ถ้าเธอออกไปก่อนหน้านี้ เธอคงทันรถบัส
Zero Conditional
If you don't water plants, they don't grow.
🇹🇭 ถ้าคุณไม่รดน้ำต้นไม้ มันก็จะไม่เติบโต
If the sun doesn't shine, plants don't make food.
🇹🇭 ถ้าแสงแดดไม่ส่อง พืชก็ไม่สังเคราะห์แสง
First Conditional
If you don't hurry, you will not catch the train.
🇹🇭 ถ้าคุณไม่รีบ คุณจะไม่ทันรถไฟ
If I don't sleep early, I won't wake up on time.
🇹🇭 ถ้าฉันไม่นอนแต่เนิ่นๆ ฉันจะตื่นไม่ทัน
Second Conditional
If I didn't have a job, I wouldn't be able to pay rent.
🇹🇭 ถ้าฉันไม่มีงาน ฉันคงไม่สามารถจ่ายค่าเช่าได้
If he weren't so shy, he would speak more.
🇹🇭 ถ้าเขาไม่ขี้อาย เขาคงพูดมากกว่านี้
Third Conditional
If he hadn't been late, he wouldn't have missed the meeting.
🇹🇭 ถ้าเขาไม่มาสาย เขาคงไม่พลาดการประชุม
If we hadn't forgotten the map, we wouldn't have gotten lost.
🇹🇭 ถ้าเราไม่ลืมแผนที่ เราคงไม่หลงทาง
ℹ️
Note: ในประโยคคำถาม ส่วน If clause ไม่เปลี่ยน แต่ main clause เปลี่ยนเป็น question form
Zero Conditional
What happens if you mix red and blue?
🇹🇭 จะเกิดอะไรขึ้นถ้าคุณผสมสีแดงกับสีน้ำเงิน?
Does ice melt if you heat it?
🇹🇭 น้ำแข็งละลายไหมถ้าคุณให้ความร้อนมัน?
First Conditional
Will you come to the party if I invite you?
🇹🇭 คุณจะมางานปาร์ตี้ไหมถ้าฉันเชิญ?
What will you do if you lose your phone?
🇹🇭 คุณจะทำอย่างไรถ้าคุณทำโทรศัพท์หาย?
Second Conditional
What would you do if you won the lottery?
🇹🇭 คุณจะทำอะไรถ้าคุณถูกหวย?
Where would you go if you could travel anywhere?
🇹🇭 คุณจะไปที่ไหนถ้าคุณสามารถเดินทางได้ทุกที่?
Third Conditional
Would you have stayed if you had known about the problem?
🇹🇭 คุณจะอยู่ต่อไหมถ้าคุณรู้เกี่ยวกับปัญหานั้น?
What would have happened if they hadn't arrived in time?
🇹🇭 จะเกิดอะไรขึ้นถ้าพวกเขาไม่มาทัน?
Quiz Time

แบบทดสอบ If Clauses

20 ข้อ ครอบคลุม 4 conditional ประเภทละ 5 ข้อ — มีคำอธิบายทุกข้อ

0️⃣ Zero Conditional • ข้อ 1–5
ความจริงสากล กฎธรรมชาติ เหตุ-ผลที่เกิดขึ้นเสมอ
ข้อที่ 1 / 20
If you ______ water below 0°C, it freezes.
เลือกกริยาที่ถูกต้อง (Zero Conditional)
✅ ถูกต้อง! ตอบ D: cool — Zero Conditional ใช้ If + V1, V1 (present simple ทั้งสองส่วน) เพราะเป็นความจริงของธรรมชาติที่เกิดขึ้นเสมอ
ข้อที่ 2 / 20
If you mix yellow and blue, you ______ green.
Zero Conditional — ผลที่เกิดขึ้นเสมอ
✅ ถูกต้อง! ตอบ C: get — Zero Conditional ใช้ present simple ทั้งสองส่วน การผสมสีเป็นกฎที่เกิดขึ้นเสมอ ไม่ต้องใช้ will หรือ would
ข้อที่ 3 / 20
ประโยค Zero Conditional ที่ถูกต้องคือข้อใด?
✅ ถูกต้อง! ตอบ B — ใช้ V1 ทั้งสองส่วน (don't get, die) เป็นกฎธรรมชาติ A ผิดเพราะใช้ will ใน if clause C คือ Third Conditional D คือ Second Conditional
ข้อที่ 4 / 20
What type is: "If you press the button, the door opens."
✅ ถูกต้อง! ตอบ A: Zero Conditional — ใช้ V1 ทั้งสองส่วน (press, opens) แสดงผลที่เกิดขึ้นแน่นอนเสมอ เป็น cause & effect
ข้อที่ 5 / 20
If you ______ too much sugar, your teeth ______.
เลือกคู่กริยาที่ถูกต้อง
✅ ถูกต้อง! ตอบ D: eat / decay — Zero Conditional ใช้ present simple ทั้งสองส่วน เป็นความจริงทางวิทยาศาสตร์ที่เกิดขึ้นเสมอ
1️⃣ First Conditional • ข้อ 6–10
อนาคตที่มีโอกาสเกิดขึ้นจริง
ข้อที่ 6 / 20
If she ______ harder, she will get a promotion.
First Conditional — เลือกรูปกริยาที่ถูก
✅ ถูกต้อง! ตอบ B: works — First Conditional ใช้ If + V1 (present simple) + will + V1 ห้ามใช้ will ใน If clause
ข้อที่ 7 / 20
If the weather ______ nice, we will go hiking.
✅ ถูกต้อง! ตอบ B: is — main clause ใช้ "will go" จึงเป็น First Conditional และ If clause ต้องใช้ V1 (present simple) คือ "is"
ข้อที่ 8 / 20
เลือกประโยค First Conditional ที่ถูกต้องที่สุด
✅ ถูกต้อง! ตอบ B — If + V1 (see), will + V1 (tell) ถูกต้องสมบูรณ์ A ผิดเพราะใช้ will ใน if clause C ผิด tense D ผิด conditional type
ข้อที่ 9 / 20
"Will you come to the party if I ______ you?"
คำถาม First Conditional
✅ ถูกต้อง! ตอบ C: invite — main clause ใช้ will come จึงเป็น First Conditional ต้องใช้ V1 present simple ใน if clause ห้ามซ้อน will
ข้อที่ 10 / 20
ประโยค First Conditional แบบปฏิเสธที่ถูกต้อง?
✅ ถูกต้อง! ตอบ C — If + don't + V1 + will + V1 ถูกต้อง A ผิดเพราะใช้ won't ใน if clause B ผิด tense D ผิด conditional type
2️⃣ Second Conditional • ข้อ 11–15
สมมติปัจจุบัน ไม่น่าเป็นไปได้
ข้อที่ 11 / 20
If I ______ a superpower, I would fly.
Second Conditional — สมมติที่ไม่เป็นจริง
✅ ถูกต้อง! ตอบ C: had — Second Conditional ใช้ If + V2 (past simple) + would + V1 เป็นการสมมติสิ่งที่ไม่เป็นจริงในปัจจุบัน
ข้อที่ 12 / 20
เลือกประโยค Second Conditional ที่ถูกต้อง
✅ ถูกต้อง! ตอบ C — ใช้ "were" (Subjunctive Mood) กับทุก subject รวมถึง I และใช้ would ใน main clause A-B ผิด tense/mood D คือ Third Conditional
ข้อที่ 13 / 20
"What ______ you do if you found a wallet on the street?"
คำถาม Second Conditional
✅ ถูกต้อง! ตอบ C: would — คำถาม Second Conditional ใช้ would ใน main clause เพราะ "found" เป็น V2 ใน if clause
ข้อที่ 14 / 20
ประโยค Second Conditional แบบปฏิเสธที่ถูกต้อง?
✅ ถูกต้อง! ตอบ B — Second Conditional ปฏิเสธใช้ If + didn't + V1 + wouldn't + V1 A ผิด tense ใน if clause C คือ Third Conditional D ผิดเพราะใช้ won't
ข้อที่ 15 / 20
If he ______ taller, he would play basketball professionally.
สมมติสถานการณ์ที่ไม่เป็นจริงในปัจจุบัน
✅ ถูกต้อง! ตอบ D: were — Second Conditional กับ "be" ใช้ "were" กับทุก subject (Subjunctive Mood) แม้ subject จะเป็น he/she/it ก็ตาม
3️⃣ Third Conditional • ข้อ 16–20
สมมติอดีตที่เปลี่ยนไม่ได้แล้ว
ข้อที่ 16 / 20
If they had arrived earlier, they ______ the concert.
Third Conditional — เลือก main clause ที่ถูก
✅ ถูกต้อง! ตอบ C: would have caught — Third Conditional ใช้ If + had + V3 ตามด้วย would have + V3 เป็นการสมมติอดีตที่เปลี่ยนไม่ได้แล้ว
ข้อที่ 17 / 20
ประโยค Third Conditional แบบปฏิเสธที่ถูกต้อง?
✅ ถูกต้อง! ตอบ D — Third Conditional ปฏิเสธใช้ "If + hadn't + V3, wouldn't have + V3" A คือ Second Conditional B ผิด grammar C ผิดโครงสร้าง
ข้อที่ 18 / 20
"Would you have stayed if you ______ about the problem?"
คำถาม Third Conditional
✅ ถูกต้อง! ตอบ D: had known — คำถาม Third Conditional ใช้ would have + V3 ใน main clause และ If clause ต้องใช้ had + V3 (past perfect) เสมอ
ข้อที่ 19 / 20
If I ______, I ______ the test.
Third Conditional — เลือกคู่กริยาที่ถูกต้อง
✅ ถูกต้อง! ตอบ B: had studied / would have passed — Third Conditional ต้องใช้ had + V3 ใน if clause และ would have + V3 ใน main clause เสมอ ไม่สามารถผสม tense ได้
ข้อที่ 20 / 20
ประโยคใดเป็น Third Conditional ที่ถูกต้อง?
✅ ถูกต้อง! ตอบ D — Third Conditional ต้องใช้ had + V3 ใน if clause และ would have + V3 ใน main clause A คือ First Conditional B คือ Second Conditional C คือ Mixed ผิด
